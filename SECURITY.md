# Security Policy

Don't use very old versions of API please.
<br>Use older versions of the API only if you are confident in your actions ⚠️

Updates: https://github.com/Nubovik01/nbvk-quotes-api/releases

| Version | Supported  |
| ------- | ---------- |
| 2.2.0   | ✅        |
| 2.1.\*  | ❌        |
| 2.0.0   | ❌        |
| 1.0.0   | ✅        |
| < 1.0.0 | ❌        |
